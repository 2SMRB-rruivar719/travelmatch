# TravelMatch
<h1 style:color:blue;>TravelMatch</h1>es una app que fusiona planificación de viajes personalizada con socialización. Permite a los usuarios conectarse con otros viajeros que tienen gustos, presupuesto y fechas similares, creando grupos de viaje adaptativos y experiencias compartidas.
